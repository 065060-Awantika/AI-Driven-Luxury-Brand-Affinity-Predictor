# Aura: Luxury Brand Affinity Predictor 
Aura is a Deep Learning application that bridges marketing strategy and data science to predict Brand Affinity. Using a multi-layer Artificial Neural Network (ANN), the model identifies "High Value Advocates" based on consumer behavior.

# Key Features
Predictive Modeling: Classifies consumers into loyalty segments using a Sequential ANN.

Advanced Architecture: Implements Batch Normalization and Dropout layers for high accuracy and model stability.

Interactive Dashboard: A real-time Streamlit interface for instant customer profiling and advocacy scoring.

# Tech Stack
Language: Python

Deep Learning: TensorFlow / Keras

Deployment: Streamlit

Environment: Google Colab / GitHub

# Dataset
The model is trained on consumer shopping trends, focusing on features like Purchase Amount (USD), Review Rating, and Previous Purchases to quantify brand loyalty.
